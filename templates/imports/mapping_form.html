How Mappings Work

Item mappings link a distributor's raw item code (from CSV exports) to a productERP catalog item.

  • Mapped — sales data for this code will be attributed to the selected catalog item.
  • Ignored — rows with this code will be silently skipped during import.
  • Unmapped — import will be blocked until this is resolved.
