Before You Import

The import tool will validate your file before writing any data. Two checks are run:

  1. Duplicate dates — if any sale dates in the file already exist for this distributor, the import is aborted.
  2. Unknown item codes — if any Item Name ID values are not mapped, the import is aborted. Go to Item Mapping to create mappings first.

Nothing is written to the database until you confirm on the preview screen.
